# AI-Powered Smart Interview Preparation System

An AI-based interview preparation platform built using Flask and Python that helps users improve interview readiness through resume analysis, ATS scoring, AI-generated interview questions, voice interaction, and answer evaluation.

---

## 🚀 Features

- Resume Upload & PDF Parsing
- Technical Skill Extraction
- ATS Resume Score Calculation
- AI-Based Interview Question Generation
- Voice Interview Support
- AI Answer Evaluation
- User Authentication System
- Modern Dashboard UI

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- PyPDF2
- Speech Recognition
- Git & GitHub
